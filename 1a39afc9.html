”Qt中使用json类“

第一部分 QJsonObject

说明：封装了JSON中的对象，可以存储多个 键值对 其中，键为字符串  类型，值为  QJsonValue  类型

(1).  创建一个QJsonobject对象

QJsonObject::QJsonObject( );

(2). 将键值对添加到QJsonObject对象中

QJsonObject::iteratou insert( const QString &key, const QJsonValue &value);

(3).  获取QJSONObject对象中 键值对的个数

int  QJsonObject::cout( ) const;
int  QJsonObject::size( ) const;
int  QJsonObject::length( ) const;

(4).  通过key得到value

QJsonValue QJsonOject::value( const QString &key ) const;
QJsonValue QJsonOject::operator[ ]( const QString &key ) const;

(5).  检查key是否存在

iterator QJsonObject::find (const QString &key);
bool QJsonObject::contains(const QString &key);

(6).  遍历key

QStingList QJsonObject::keys ( ) const;

第二部分 QJsonArray

说明：封装了json中的数组  数组中元素为QJsonValue类型

(1).  创建一个QJsonArray

QJsonObject::QJsonArray( );

(2). 添加数组元素

void QJsonArray::append(const   QJsonValue &value )；//添加到头部
void QJsonArray::prepend(const  QJsonValue &value )；//添加到尾部
void QJsonArray::instert ( int i, const QJsonValue &value);//插入到 i 的位置之前
void QJsonArray::push_back ( const QJsonValue &value);//添加到头部
void QJsonArray::push_front ( const QJsonValue &value);//添加到尾部

(3). 获取QJsonArray中元素的个数

int QJsonArray::count( ) const;
int QJsonArray::size( ) const;

(4). 获取元素的值

QJsonValue QJsonArray::first( ) const;//获得头部
QJsonValue QJsonArray::last( ) const; //获得尾部
//获得指定位置的值
QJsonValue QJsonArray::at( int i ) const;
QJsonValue QJsonArray::operator[ ]( int i ) const;

(5). 删除元素

void QJsonArray::pop_back( )；      //删除头部
void QJsonArray::pop_front()；      //删除尾部
void QJsonArray::removeFirst( )；   //删除头部
void QJsonArray::removeLast()；     //删除尾部
//删除指定位置
void QJsonArray::removeAt ( int i );
void QJsonArray::takeAt( int i );

第三部分 QJsonValue

(1). 封装json支持6种数据类型

QJsonValue::Bool          //布尔类型
QJsonValue::Double        //浮点类型（包括整形）
QJsonValue::String        //字符串类型
QJsonValue::Array        //json数组类型
QJsonValue::Object        //json对象类型
QJsonValue::Null        //空值类型

(2). 构造QJsonValue对象

//字符串
QJsonValue ( const char *s );
QJsonValue ( QLatinlString s );

//整形 and 浮点形

//布尔类型

//json数组类型

//json对象类型

//空值类型
QJsonValue（ QJsonValue::Type type = NULL );

(3). 判断一个 QJsonValue对象 的数据类型

bool isString:( ) const;//是否是字符串类型
bool isDouble:( ) const;//是否是浮点类型
bool isBool:( ) const;//是否布尔类型

第四部分 QJsonDocument

说明：封装了一个完整的JSON文档

以二进制的方式读取或写入文档  以utf-8形式表示

QJsonObject和QjsonArray不能直接转换为字符串类型 需要通过QJsonDocuemt类来进行转换

(1). QJsonObject / QjsonArray => 字符串

a. 创建QJsonDoucument 对象  以QJsonObject 或 QJsonArray为参数创建QJsonDoucunt 对象

QJsonDocument::QJsonDoucument( const QJsonObject &object);      //第二个为定义的对象名字QJsonDoucument
QJsonDocument::QJsonDoucument( const QJsonArray &object);       //第二个为定义的对象名字QJsonDoucument

b . 将QJsonDoucument对象中的数据进行序列化

 通过调用toxxx（）方法可以得到文本格式或者二进制格式的JSon字符串

QByteArray QJsonDocument::toBinaryData( ) const;
QByteArray QJsonDocument::toJson( ) const;

c. 使用得到的字符串进行数据传输，或者保存到文件

(2).字符串 => QJsonObject / QjsonArray

a. 将 JSON字符串转换为 QJsonDoucument 对象

[static] QJsonDocument  QJsonDoucument::fromBinaryData( const QJsonArray &data, Datavalidation validation = validation);
[static] QJsonDocument  QJsonDoucument::fromjson( const QJsonArray &json,QJsonParseError * error = Q_NULLPTR);

(3). 将QJsonDoucument对象 转换为json数组 /对象

a. 判断文档对象中存储的数据，是JSON 数组还是JSON 对象

bool QJsonDocument::isArray:() const;

bool QJsonDocument::isObject() const;

b. 之后就可以转换为数组或JSON对象

QJsonObject QJSONDocument::object:( ) const;

QJsonArray QJsonDocument::array( ) const;

c. 调用QJsonArray / QJsonObject 类提供api获取存储在其中的数据
